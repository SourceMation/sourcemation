Details

This is the minimal (“slim”) variant of the Sourcemation Docker image for Debian 13, codenamed “Trixie”, which is the current stable release of the Debian operating system. It’s built from scratch, and stripped of all non-essential packages—like documentation, locales, but also common packages like tzdata or even ca-certificates - to create the smallest, but fully functional base image possible. This results in a significantly reduced footprint, enhanced security, and faster container deployments, making it an ideal base image that we are internally using for many of our containerized applications that require a stable and modern OS layer.
