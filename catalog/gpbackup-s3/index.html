GPBackup-S3

Details

GPBackup-S3 is a tool for automatic backups of Greenplum Database with Amazon S3 integration. It enables easy storage, restoration, and verification of full and incremental backups in a secure, scalable cloud. This is an ideal solution for organizations looking to efficiently manage data in the cloud.
